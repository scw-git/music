### 功能比较齐全的音乐播放器
模仿网易云的音乐播放器，实现了基本功能。还有一些功能，没有一一展示，后续还会增加一些功能。
主要的技术难点，就是实现音乐的播放，还有歌曲滚动。用到了$emit $on $parent等传值方式。
### 目前功能（后续还会添加）
* 全部歌单
* 歌单详情
* 最新音乐
* 最新mv
* 歌曲评论、mv评论
* 相似mv
* 搜索详情
* 自动播放
* 歌曲详情页
